Student Guides

Welcome to Your Git & GitHub Adventure!

Ready to level up your coding skills? This guide will help you learn how to use Git, GitHub, and VS Code like a pro. Let’s get started!

Why Use Git?

Think of Git as a magic save button for your code. It lets you go back in time, try new ideas, and work with friends—without losing your work!

Getting Started with VS Code

  1. Download and install VS Code.
  2. Open your project folder in VS Code.
  3. Click the Source Control icon on the left sidebar.
  4. Click Initialize Repository to start using Git!
Pro Tip: Try out different VS Code themes to make your editor your own!

Git Basics for Students

  • Repository: Your project folder tracked by Git.
  • Commit: A snapshot of your work (like saving your game!).
  • Branch: A separate path for new ideas—experiment without breaking your main project.
  • Push: Send your changes to GitHub so your teacher can see them.
Try It! Make a change in your code, then commit and push it. Can you see your update on GitHub?

Using GitHub in Class

  1. Sign up at github.com (ask your teacher if you need help).
  2. Accept your assignment link from your teacher.
  3. Clone the repository to your computer using VS Code.
  4. Work on your code, commit your changes, and push them to GitHub.
Did You Know? You can add emojis to your commit messages! Try :sparkles: or :bug:.

Teamwork & Collaboration

  • Work on different branches to avoid conflicts.
  • Open a Pull Request to suggest changes.
  • Review your classmates’ code and leave positive feedback!
Collaboration Challenge: Pair up and try to merge your code together. Can you resolve a merge conflict?

Troubleshooting & FAQ

  • Accidentally deleted a file? Use git checkout -- <filename> to bring it back.
  • Can’t push? Make sure you’ve committed your changes and are connected to the internet.
  • Merge conflict? Ask a friend or your teacher for help—everyone gets them sometimes!
Tip: Don’t panic! Mistakes are part of learning. Use Git to experiment safely.
